# Futr Loading V2 

**FREE** advanced loading screen for FiveM with YouTube video support

##  Features

-  **YouTube Video Support** – Play videos directly from YouTube as your loading screen background
-  **Modern & Clean Design** – Professional interface with smooth animations
-  **Performance Optimized** – Lightweight and efficient resource usage
-  **Easy Configuration** – Simple setup and customization options
-  **Responsive Design** – Works well across different screen sizes

##  Installation

1. Download the latest release from the [Releases](#) section
2. Extract the folder to your FiveM server resources directory
3. Add `ensure futr-loadingv2` to your server.cfg file
4. Configure settings in the configuration file
5. Restart your server
